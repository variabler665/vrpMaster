# vrpMaster

## Быстрый запуск симулятора

Теперь не нужно вручную поднимать сервер. Просто запустите файл `start_sim.py`
(двойным кликом или командой ниже), и страница автоматически откроется в
браузере:

```bash
python3 start_sim.py
```

Скрипт сам поднимет локальный HTTP-сервер, подберёт свободный порт (начиная с
`8000`) и откроет `http://localhost:<порт>/` с интерфейсом симулятора. Остановить
сервер можно сочетанием `Ctrl+C` в окне, где запущен скрипт.

Дополнительно:
- Ключ `--port <номер>` фиксирует порт, если нужно своё значение.
- Ключ `--no-browser` отключает автоматическое открытие браузера.
